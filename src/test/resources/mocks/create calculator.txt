To create a Maven project named `calculator` that implements a Java Swing calculator application,
I will perform the following actions:

1. **Create the project directory structure**
2. **Generate the Maven project files** (pom.xml)
3. **Create the Java source files** for the calculator application
4. **Create the Swing GUI class**

Let's start by creating the necessary directories and files.
